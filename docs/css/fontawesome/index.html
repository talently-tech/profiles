Font Awesome v5

Aquí verás como usar Font-Awesome 5. Si miras el código de este archivo podrás ver instrucciones.

Como los uso

  1. Debes copiar la carpeta completa de Fontawesome a tu proyecto
  2. En el footer de la carpeta agregas la etiqueta script que llamará al archivo de javascript que genera los íconos: <script defer src="fontawesome-free-5.8.2-web/js/all.min.js"></script>

  3. Para usarlos tienes que crear un elemento HTML de tipo i y agregarle dos clases, fasen el caso de iconos genéricos y fab cuando son íconos de marcas; luego otra con el prefijo fa- seguido del nombre del ícono por ejemplo: 

                            
                            
                            <i class="fas fa-user"></i>
                            <i class="fab fa-facebook"></i>
